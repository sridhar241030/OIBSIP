#  PizzaVerse - Full Stack Pizza Delivery Website

A modern full-stack Pizza Delivery web application developed using React, TypeScript, Express.js, and Supabase.

This project was developed as part of the **Oasis Infobyte Level 3 Web Development & Designing Internship**.

---

#  Features

##  User Features

- User Registration & Login
- Browse Pizza Menu
- Search Pizzas
- Filter Pizzas
- Add to Cart
- Update Cart Quantity
- Secure Checkout
- Place Orders
- Order History
- Customer Reviews
- Responsive Design

##  Admin Features

- Admin Dashboard
- Add New Pizza
- Update Pizza Details
- Delete Pizza
- Manage Inventory
- Manage Orders
- Manage Customer Reviews

---

#  Tech Stack

## Frontend

- React
- TypeScript
- Vite
- TanStack Router
- Tailwind CSS

## Backend

- Node.js
- Express.js

## Database

- Supabase

## Payment Gateway

- Razorpay (Test Mode)

#  Installation

## 1. Clone Repository

```bash
git clone https://github.com/sridhar241030/OIBSIP.git
```

## 2. Open Project

```bash
cd OIBSIP/WebDev-L3-PizzaDelivery
```

## 3. Install Frontend Dependencies

```bash
npm install
```

## 4. Install Backend Dependencies

```bash
cd server
npm install
```

---

#  Environment Variables

Create the following files:

Frontend

```text
.env
```

Backend

```text
server/.env
```

Use the values provided in:

- `.env.example`
- `server/.env.example`

# ▶ Run Frontend

```bash
npm run dev
```

# ▶ Run Backend
